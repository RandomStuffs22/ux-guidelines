Breadcrumb Pattern

A navigation aid to keep track location in an application or website

Overview

Breadcrumb is a navigation aid used in user interfaces. It allows users to keep track of their locations within programs or documents.
The main purpose of the document is to provide a pattern of breadcrumbs.

What Problem Does This Solve?

The user needs to be able to navigate up (towards the root page) and have an understanding of where he is in relation to the rest of the site.

When to Use This Pattern?

Breadcrumbs should be used when the structure of the website follows a strict hierarchical structure of similar formatted content. Especially, when the structure of the site is parted into sections which can be divided into more subsections and so on. In case the user cannot easily navigate through the hierarchy via other local navigation methods, breadcrumbs is useful.

What's the Solution?

Template

The standard breadcrumb template:

  • Home
  • Home /
  • Library
  • Home /
  • Library /
  • Data

Markup

HTML is your standard unordered list with links.

    

  1. <ul class="breadcrumb">
  2.  <li>
  3.    <a href="#">Home</a> <span class="divider">/</span>
  4.  </li>
  5.  <li>
  6.    <a href="#">Library <span class="divider">/</span>
  7.  </li>
  8.  <li class="active">Data</li>
  9. </ul>

Status

Status Description
Standard Status The labels of the sections in the hierarchical path that can be clicked to go to the home or previous/parent page of the being viewed page.
Being Viewed Status The label of the current page is at the end of the breadcrumb and is not linked.

Rules

BR-01 Non breadcrumbs at the home page
BR-02 Breadcrumbs is on the top of a page content
BR-03 Breadcrumb label is the page content title
BR-04 Breadcrumbs label length is limited with 20 words
BR-5 Hyperlink all label except the last one and breadcrumbs icon
BR-06 Hyperlink should be styled the same regardless of whether it has been visited or no
BR-07 Breadcrumb box must cover the breadcrumbs path
BR-08 Breadcrumb length must be auto resized and have a same minimize limitation in compare with its site
BR-09 Breadcrumb broken in case it is longer than the minimize limitation
BR-10 There are 2 spaces before and after each breadcrumb divider

Why to Use This Pattern?

Because of 3 main benefits of breadcrumbs:

  • Provide users information where they are in a site/application
  • Offer shortcut links for users to “jump” to previously viewed pages
  • Allow users know the structure of a site/application
    • It should be used in all sub pages. Besides, the consistency of breadcrumbs in use will make eXo products look more professional and statistical.
