File structure

Within the download you'll find the following file structure and contents, logically grouping common assets and providing both compiled and minified variations.

Once downloaded, unzip the compressed folder to see the structure of eXo UX Guidelines (based on Bootstrap). You'll see something like this:

  exo-ux-guidelines /
  ├── css/
  │   ├── bootstrap.css
  │   ├── bootstrap.min.css
  │   ├── plf4.css
  ├── js/
  │   ├── bootstrap.js
  │   ├── bootstrap.min.js
  ├── img/
  │   ├── icon_start.png
  │   ├── icon_pattern.png
  │   ├── plf4-icons
  └── README.md

All CSS, JS files are based on (bootstrap.*), as well as compiled and minified CSS and JS (bootstrap.min.*).

(plf4.css) is the customized Bootstrap CSS that is applied for the eXo Platform, done by The eXo Platform Team.

Please note that most JavaScript plugins require jQuery to be included. Besides, there some free javacripts are also applied in eXo UX Guidelines to create some dynamic samples.

Basic HTML template

eXo UX Guidelines has basic HTML templated provided by Bootstrap.

Typical HTML file:

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.   </head>
  6.   <body>
  7.     <h1>Hello, world!</h1>
  8.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  9.   </body>
  10. </html>

To make this a Bootstrapped eXo Platform style template, just include the appropriate CSS and JS files of Bootstrap and the customized CSS that is provided by The Platform Team.

Please note that the plf4.css is the Core.css file that is located in skin/css of eXoResource.war in the Platform 4 package. You can copy, rename and put it in the general css folder that contains Bootstrap CSS. 

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.     <!-- Bootstrap -->
  6.     <link href="css/bootstrap.css" rel="stylesheet">
    <!-- Platform 4 CSS -->
  7.     <link href="css/plf4.css" rel="stylesheet">
  8.   </head>
  9.   <body>
  10.     <h1>Hello, world!</h1>
  11.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  12.     <script src="js/bootstrap.min.js"></script>
  13.   </body>
  14. </html>

And you're set! With those files added, you can begin to develop any site or application with eXo Platform Style based on Bootstrap.
