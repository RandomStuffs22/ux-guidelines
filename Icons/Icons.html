Icons

Style

Color

Sample Title Color Overlay Apply For
Light Gray Icon #999999 Default color for repeated action icons, action list having dark gray labels. E.g: Delete or Edit action in table, menu drop down with dark gray text.
Dark Gray Icon #626262 Default color for action icons having light or medium gray labels.
White #FFFFFF Hover icon in dropdown menu, icons in primary button or black/gray bar.
Electric Blue #4F72AC Default color icon for hover/pressed/selected effect on Light or Dark Gray Icon.

Platform 4 icons are located in sprites. Each sprite has 4 color version. For example, icons in the general sprite has 4 colors that are define by 4 classes: uiIconLightGray, uiIconDarkGray, uiIconWhite, uiIconBlue.

Size

Sample Size Apply For
32x32 Space application bar, special purpose in use.
24x24 Drive, top navigation icons, special purpose in use.
16x16 Default icon for inpendent use such as icon button or icon with 13px label.
14x14 Icon with 12 or 11px font label or special hover action icons.
9x9 Special icon in use such as arrow, close...
7x7 Special icon in use such as mini arrow...

How to use

All icons require an <i> tag with a unique class, prefixed with uiIcon. To use, place the following code just about anywhere:

<i class="uiIconTrash"></i>

There are also styles available for inverted (light gray, dark gray, white, electric blue) icons, made ready with one extra class. We will specifically enforce this class on hover and active states for nav and dropdown links.

<i class="uiIconTrash uiIconLightGray"></i>
<i class="uiIconTrash uiIconDarkGray"></i>
<i class="uiIconTrash uiIconWhite"></i>
<i class="uiIconTrash uiIconBlue"></i>
