Sort By Column Pattern

The user needs to search or scan a table for values that is of interest. How to sort data by column?

Overview

Sort by column is to arrange the data display depending on the criteria of a specific column. The main purpose of the document is to provide a pattern of sort by column.

What Problem Does This Solve?

The users may want to sort data by specific criteria to find the expected information easier.

When to Use This Pattern?

Sort by column can be used in case the data is displayed in different columns.

What's the Solution?

Status

Status Description
Standard column The title of this column is not used as criteria for sorting
Being sorted column The title of this column is used as criteria for sorting. There is up or down icon that represents for direction of the sort.

Rules

SR-01 No sort by column in case 0 or 1 data row
SR-02 Up or down icon displays at the being sorted column
SR-03 Column label matches with the data column
SR-04 Column label length is limited with 20 words

Why to Use This Pattern?

To provide an aid for users to sort data and make the consistency among products in the way sorting and displaying data.
