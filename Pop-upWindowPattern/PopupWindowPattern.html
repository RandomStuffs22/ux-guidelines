Modal Pop-up Pattern

Using popups can be useful and a nice way to display advanced information. How and when Popups is used with PLF 4 style?

Overview

Pop-up is a form displays when selecting an option or doing an action by mouse click or key board press.
Pop-up can be used to perform specific functions or simply notify necessary information to users.
This pattern provides a guide to use and display modal pop-up to obtain the consistency among applications.

What Problem Does This Solve?

Pop-up is useful and fast way to interact with a function, it is also to display advanced options over one element on the screen without having to lose the content.

When to Use This Pattern?

Use when a content is suitable to display in a pop-up.
Data input and display are short
Interact with simple function
Common action that is used by different application
Can be grouped in different tab logically
Use when there is expectation to keep current page while interacting with more functions.

What's the Solution?

Pop-up Size

Modal provides a pop-up with fixed width 560px. The height of pop up is automatic resized depending on the content and height of the screen. There is a scroll bar in case the content is longer than the display screen.

Why to Use This Pattern?

To obtain the consistency among applications in the way using and displaying actions/functions in pop-ups. This pattern let the designers know which functions they should put in page, which functions should be in pop-up. The consistency of pop-up style sheet and design makes your products look more professional.
