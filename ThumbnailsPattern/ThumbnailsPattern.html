Overview

Thumbnails are reduced-size images that make easy to recognize and organise them. Thumbnails can be used to display pictures in a gallery, image attachments, files in a folder, avatars... This page is mainly on Image Attachment Thumbnail and Thumbnail in a Folder.

What Problem Does This Solve?

The user needs to get an overview of multiple pictures without having to download each of the full size images.

When to Use This Pattern?

Thumbnails can be used to present images in cases image attachments in topic, question and files that are displayed as thumbnails in a folder.

What is the Solution?

Image Attachment Thumbnails

The thumbnails with rounded borders can be used for image attachments in a topic, question... the height is fixed 80px, the width is resized automatically based on the original size of images.
