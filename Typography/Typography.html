Headings

Typography is about typefaces, such as font style, font size, color and space around and through them to create a complete design.
Good typography makes good design for an application. Each typeface can be used for specific purposes that help to recognize role or importance of specific content.
Each application has a common typography that shows its own style and also obtains consistency and professionalism. Therefore, the typography should be applied into the whole application.
All HTML headings, from <h1> to <h5>, are available. Besides, the cover or page title can be defined by large font size.

All headings are defined in bootstrap.css which is customized by exo-platform.css. In particular:

Heading Font Size Apply For
.hero unit h1 60px Cover Titles, Page Titles, Product Titles
h1 36px Top level headings, Main subjects
h2 30px Important subjects
h3 24px Sub main important subjects
h4 18px Main headings of a sub important subject
h5 14px Sub content headings of a sub important subject

In some cases, the font size can be smaller depending its purpose:

Font Size Apply For
13px Default font size for body copy, default menu, default button.
12px Minor content notice, used for specific purposes.
11px Mini text content, used for specific purposes.
Font Color Title Apply For

#333333

 Dark Gray Default color for all titles, texts, text input, and labels.

#808080

 Medium Gray Text color in navigations, used for special purposes

#999999

 Light Gray Default gray color for small texts, sub-titles, and text explanations.

#2F5E92

 Electric Blue Emphasized titles, text hover or press, hyper-links or selected links.

font-family "Helvetica Neue", Helvetica, Arial, sans-serif
