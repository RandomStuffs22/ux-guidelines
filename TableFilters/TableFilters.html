Table Filter Pattern

The user wants to narrow down the search results shown in a table by specific columns. How to filter it?

Overview

Table filters allow to search data in a table with specific conditions. The main purpose of the document is to provide a pattern of table filter.

What Problem Does This Solve?

The user may want to filter the search results shown in a table by specific columns.

When to Use This Pattern?

Table filter is useful in case there is large data set of result that can be filtered to find expected information. Besides, there should be one or more table column can be grouped into categories to filter by.

What's the Solution?

There is a select box or text field input that allow to define the condition value to filter in each column. When a filter condition is selected, the data in table is filtered automatically. Optionally, multiple filters can be added. For example, 2 or 3 filters are selected at once, the results return in the table must be satisfied all filter conditions.

Status

Status Description
Default status The table displays with default data that is all data or filtered by default conditions.
Filtered status Users execute a specific filter .The table displays with filtered data.

Rules

TFR-01 Column data matches with its filter
TFR-02 Filters are displayed below the column title
TFR-03 Filter text length is limited with 20 words
TFR-04 Data can be filtered by condition combination

Why to Use This Pattern?

Because of 2 main reasons: Filters reduce the amount of items display and Filters help to narrow down search result that make users find expected results easily.
Besides, the same usage makes the consistency among products in the way filtering and displaying data.
