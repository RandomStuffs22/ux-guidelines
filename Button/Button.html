Default Button

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

Button class="" Description Apply For
btn Standard gray button with gradient Normal action buttons such as Add New, Edit, Delete...
btn btn-primary Provides extra visual weight and identifies the primary action in a set of buttons Primary action button expected to do such as Save, Ok, Accept...
btn btn-info Used as an alternative to the default styles Buttons on information page or pop-up such as Ok, View details...
btn btn-success Indicates a successful or positive action Buttons on successful information page or pop-up such as Ok, Share...
btn btn-warning Indicates caution should be taken with this action Button on warning page or pop-up such as Ok, More details...
btn btn-danger Indicates a dangerous or potentially negative action Button on error page or pop-up such as Ok, Help, View details...
btn btn-inverse Alternate dark gray button, not tied to a semantic action or use In case inverse color is applied for a page or pop-up, inverse buttons can be applied correspondingly

Button Size

Fancy larger or smaller buttons? Add .btn-large, .btn-small, or .btn-mini for additional sizes.

Button class="" Description Apply For
btn btn-large Font size: 15px Action buttons that expected to attrach attention such as Join with Us, View detail for hot update, advertises...
btn Font size: 13px
Line-height: 18px		 Default button for all actions in page or pop-up.
btn btn-small Font size: 11px
Line-height: 16px		 A function may include some sub-functions. With sub-function in form, small buttons can be used.
btn btn-small Font size: 11px
Line-height: 14px		 Hover reveal actions, repeated actions in a table

Create block level buttons—those that span the full width of a parent— by adding .btn-block.

<button class="btn btn-large btn-block btn-primary" type="button">Block level button</button>
<button class="btn btn-large btn-block" type="button">Block level button</button>

Depending on special goals, the block level button can be used to draw user attention to do specific action.

Disabled state

Make buttons look unclickable by fading them back 50%.

The disabled state can be used in case some action buttons are not available. E.g the Next button is disabled until the current input form is completed.

Anchor element

Add the .disabled class to <a> buttons.

Link 

<a href="#" class="btn btn-large disabled">Link</a>

Button element

Add the disabled attribute to <button> buttons.

<button type="button" class="btn btn-large" disabled>Button</button>

One class, multiple tags

Use the .btn class on an <a>, <button>, or <input> element.

Link 
	<a class="btn" href="">Link</a>
	<button class="btn" type="submit">Button</button>
	<input class="btn" type="button" value="Input">
	<input class="btn" type="submit" value="Submit">

As a best practice, try to match the element for your context to ensure matching cross-browser rendering. If you have an input, use an <input type="submit"> for your button.
